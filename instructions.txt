To run through the PocketIC demo, follow the following steps:

Make sure to set up the sns-testing tooling:

$ ./install.sh

and download [SRC wasm module](https://github.com/dfinity/subnet-rental-canister/releases/download/0.1.0/subnet_rental_canister.wasm).

1. Run the PocketIC server

$ curl https://download.dfinity.systems/ic/05cbe03829ab40e00c4baee577a354e4e9bafaad/binaries/x86_64-linux/pocket-ic.gz -o pocket-ic.gz
$ # curl https://download.dfinity.systems/ic/05cbe03829ab40e00c4baee577a354e4e9bafaad/binaries/x86_64-darwin/pocket-ic.gz -o pocket-ic.gz
$ gzip -d pocket-ic.gz
$ chmod +x pocket-ic
$ ./pocket-ic --port 8000 --ttl 3600

2. In a separate tab, create an IC instance with NNS, SNS, and application subnet.

$ ./pocket_ic.sh

3. In a separate tab, set up NNS, and run the basic SNS testing scenario.

$ ./cleanup.sh && ./setup_locally.sh

You can open the NNS frontend dapp in your browser at

http://qsgjb-riaaa-aaaaa-aaaga-cai.localhost:8080/
