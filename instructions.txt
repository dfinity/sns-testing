To run through the PocketIC demo, follow the following steps:

The first step should be executed from the root of the dfinity/ic repository.

1. Run the PocketIC server

$ ./gitlab-ci/tools/docker-run
$ bazel run //rs/pocket_ic_server:pocket-ic-server -- --port 8000 --ttl 3600

The remaining steps should be executed from the root of the dfinity/sns-testing repository.

2. Create an IC instance with NNS, SNS, and application subnet.

$ ./pocket_ic.sh

3. Download and run icx-proxy.

$ wget https://download.dfinity.systems/ic/f2cd225b621674b19f3d601ebb42555d0c6f614d/binaries/x86_64-linux/icx-proxy-dev.gz
$ gzip -d icx-proxy-dev.gz
$ chmod +x icx-proxy-dev
$ ./icx-proxy-dev --replica http://localhost:8000/instances/0/ --address [::]:8080 --fetch-root-key

4. Set up the sns-testing tooling.

$ ./install.sh

5. Set up NNS and run the basic SNS testing scenario.

$ ./cleanup.sh
$ ./setup_locally.sh
$ ./run_basic_scenario.sh

You can open the NNS frontend dapp in your browser at

http://qsgjb-riaaa-aaaaa-aaaga-cai.localhost:8080/
